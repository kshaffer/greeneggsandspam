Monkeys Writing Shakespeare (or Seuss...)

Following is the text of Dr. Seuss's Green Eggs and Ham. But with a twist... Every so often, a word will be replaced with another word that rhymes with it.

How long will it take the random word changes to repeat the original text?

This site uses a JavaScript app that chooses a random word in the text and searches for it in custom lists of rhyming words. If the word is found in one of the lists, it is replaced by another word in the list. Then another random word is chosen, ad infinitum. Or stoppum pushum. Or ad tabbus closium. :)

You can find (and copy and edit) the code for this project on GitHub.
